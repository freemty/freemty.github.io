Abstract

Proposed Model

为了方便阅读，我们先记

  • discriminator \(D\)
  • generator \(G\)
  • vocabulary \(\gamma\)
  • a sequence \(Y_{1:T} = (y_1,...,y_t,...,y_T),y_t ∈ \gamma\)
  • current state \(s = Y_{1:t-1}\)(当前的状态为之前所有输出tokens的序列)
  • next state \(s^{,} = Y_{1:t}\)
  • action \(a = y_t\)

在上面的预设下，所有Text-Generation类任务，都可以抽象化为在\(t\)时刻，已经生成\((y_1,..y_{t-1})\)的基础上，采取什么action（如何选取\(y_t\)）。
